**Props**

| Prop           | Type                    | Default    | Description                               |
|----------------|-------------------------|------------|-------------------------------------------|
| `username`     | `string`                | `""`       | your leetcode username                    |
| `theme`        | `string`                | `"green"`  | `"green"`, `"blue"`, `"red"`, `"orange"`, `"purple"`, `"yellow"` |
| `blockVariant` | `string`                | `"default"`| `"default"`, `"outline"`                 |
| `blockSize`    | `string`                | `"default"`| `"sm"`, `"default"`, `"lg"`, `"xl"`      |
| `blockAnimate` | `string`                | `"none"`   | `"none"`, `"pulse"`, `"bounce"` (only on high activity days) |
| `variant`      | `string`                | `"default"`| `"default"`, `"shadow"`                  |
