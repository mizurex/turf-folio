### Props

| Prop     | Type              | Description                                      |
|----------|-------------------|--------------------------------------------------|
| `stores` | `PriceMapStore[]` | Array of stores. Each has `id`, `lat`, `lng`, `price`, `address`. |

### Usage

```tsx
import PriceMap from "@/docs/registry/components/price-map";

const stores = [
  { id: 1, lat: 28.6139, lng: 77.209, price: "₹3.50", address: "Shop 15, Connaught Place, New Delhi" },
  { id: 2, lat: 19.076, lng: 72.8777, price: "₹5.50", address: "Unit 205, R City Mall, Mumbai" },
];

<PriceMap stores={stores} />
```
